# C-quote
**intellisense suggestion**(*super powerful*)
# C-;
toggle line comment
# M-;
toggle region comment
# M-^
Merge the previous and the current line(**regardless of the position of the cursor**)
# M-g n
jump to next error
# M-g p
jump to previous error
# M-%
replace
# M-y
yank pop
# C-M-spc
toggle sidebar visibility
# C-x z
toggle zen mode
# C-x C-w
save as
# C-x h
select all
# C-S-backspace
kill whole line
